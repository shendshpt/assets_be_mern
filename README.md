# Getting Started with Create React App

This project MERN stack

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the server only exclude nodemon.\

---

### `npm run server`

Runs the server only with nodemon.\

---

### `npm run client`

Run react app only with connection: http://localhost:3000

---

### `npm run dev`

Run Server and Client with one script

---

### `npm run data:import`

Run script for import data from local folder to database, syntax in file seeder.js

---

### `npm run data:destroy`

Run script for delete data from database, syntax in file seeder.js
